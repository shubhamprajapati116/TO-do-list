# Todo List App
A simple and responsive Todo List App to add, mark as completed, and delete tasks. Tasks are saved in the browser's local storage so they remain even after refreshing the page.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Local Storage
